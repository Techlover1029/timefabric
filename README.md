# TimeFabric

**TimeFabric** is a Windows-first, source-available timecode engine designed for professional live production, broadcast, and media workflows.  
It generates, translates, and routes multiple independent timecode clocks across protocols such as MTC, OSC, Art-Net, sACN, NDI, and PTP using a modular, node-based architecture.

TimeFabric supports:
- Multiple independent clocks with frame-accurate timing
- Free routing between inputs and outputs
- Drop-frame and non-drop-frame SMPTE support
- Smooth chase and user-configurable failure policies
- Professional integration for lighting, broadcast, and media production
